# EDev Portfolio

Portafolio profesional de Emmanuel Palma Solís.

## Descripción

Sitio web desarrollado para presentar proyectos, experiencia, formación y habilidades técnicas relacionadas con desarrollo Android, desarrollo web, automatización y soluciones digitales.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Responsive Design
* Git
* GitHub Pages

## Proyectos destacados

* DepotLink POS Mobile
* PepsOCR
* Smart POS
* Taquería POS Android
* EDEV Analytics

## Características

* Diseño responsive
* Optimizado para dispositivos móviles
* Navegación dinámica
* Galería de proyectos
* Descarga de CV
* Portafolio profesional
